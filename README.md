# cloudPicture
基于Springboot+Vue+COS+WebSocket的云图床

![image-20250104153527510](cloud-picture-frontend\public\6778e4eae9726.png)

### 技术选型：

<hr>

#### 后端

* Springboot
* MySQL数据库+Mybatis-Plus框架
* Redis缓存+Caffeine本地缓存
* jsoup抓取数据

#### 前端

* Vue3框架
* Vite构建工具
* Ant Design Vue组件库
* Pinia全局状态管理
* ESLint+TypeScript
